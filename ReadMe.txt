The files in this archive are presented to help work with the code presented in the book. 

Errors should be reported to philo89@msn.com.

This source code includes templates for InfoPath forms. If you wish to inspect the contents of the template before opening the template in InfoPath, rename the xsn file to have an extension of cab. Then you can open the archive directly in Windows XP or 2003 (or use an archive manager like WinZip). All the files contained within are plain text files and can be opened in any text editor. 

The folders with projects include only source code files, no compiled binaries.
